# Configuration Wizard, Configuration Complete Page
Use the **Configuration Complete** page to review the features and result of the BizTalk Server configuration process.  
  
|Use this|To do this|  
|--------------|----------------|  
|**Logfile**|Click **Logfile** to view the configuration log file. This file is stored in your temp directory.|  
|**Finish**|Click **Finish** to close the Configuration Wizard.|
