# Configuration Wizard, Service Accounts Page
Use the **Service Accounts** page to edit the selected service account.  
  
|Use this|To do this|  
|--------------|----------------|  
|**User name**|Type the user name for this service account.|  
|**Password**|Type the password for the account.|
