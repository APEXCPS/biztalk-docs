# MSBTS_HostSetting.DecryptCertComment Property (WMI)
Represents a comment field that enables you to associate a friendly name with a decryption certificate.  
  
> [!NOTE]
>  The syntax shown is language neutral.  
  
## Syntax  
  
```  
  
string DecryptCertComment;  
```  
  
## Remarks  
 This property is read-write.  
  
 The maximum length for this property is 256 characters.  
  
 For sample code using the **MSBTS_HostSetting** class, see [Creating, Updating, and Deleting a Host Instance Using WMI](../core/creating-updating-and-deleting-a-host-instance-using-wmi.md).  
  
## Requirements  
 **Header:** Declared in BTSWMISchemaXP.mof.  
  
 **Namespace:** Included in \root\MicrosoftBizTalkServer.
