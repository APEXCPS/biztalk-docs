# SStatus Fields
The fields of the **SStatus** interface are listed in the following table. For a complete list of **SExternal** interface members, see [SStatus Members](../core/sstatus-members.md).  
  
## Public Fields  
  
|Field|Description|  
|-----------|-----------------|  
|[ullErrorCode](../core/sstatus-ullerrorcode-field.md)|An integer that contains an error code.|  
|[bstrErrorMessage](../core/sstatus-bstrerrormessage-field.md)|A string that contains an error message.|
