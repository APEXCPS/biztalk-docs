# IBTTransportConfig Interface (COM)
Defines the methods that all receive adapters need to implement in order to receive configuration information from the BizTalk Server Messaging Engine.  
  
 For a list of all members of this type, see [IBTTransportConfig Members](../core/ibttransportconfig-members-com.md).  
  
> [!NOTE]
>  For information on using this interface within managed code, see `IBTTransportConfig Interface`.
