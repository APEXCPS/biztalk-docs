# Define a Constant Value Page
Use the **Define a Constant Value** page to specify a name, type, and constant value for the definition.  
  
|Use this|To do this|  
|--------------|----------------|  
|**Definition type**|Select a .NET-based type for the definition from the drop-down list.|  
|**Display name**|Specify a name for the vocabulary definition for display in rule definitions.|  
|**Value**|Type a definition value that is valid for the selected .NET-based type.|
