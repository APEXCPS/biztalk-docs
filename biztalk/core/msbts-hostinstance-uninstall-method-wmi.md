# MSBTS_HostInstance.Uninstall Method (WMI)
Uninstalls a service for the given instance of the BizTalk host.  
  
## Method Declaration  
 *The syntax shown is language neutral.*  
  
```  
uint32 Uninstall();  
```  
  
## Return Value  
 This method returns an HRESULT indicating whether the method completed successfully.  
  
## Remarks  
 For samples illustrating the **MSBTS_HostInstance** class, see [WMI Script Samples](../core/wmi-script-samples.md).  
  
## Requirements  
 **Header:** Declared in BTSWMISchemaXP.mof.  
  
 **Namespace:** Included in \root\MicrosoftBizTalkServer.
