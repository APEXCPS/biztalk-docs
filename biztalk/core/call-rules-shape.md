# Call Rules Shape
You can use the **Call Rules** shape to specify a rule policy to be applied using selected orchestration variables as facts.  
  
|Use this|To do this|  
|--------------|----------------|  
|**Configure Policy** property|Specify the policy to be executed by the rule engine. Click the ellipsis (**â€¦**) button to invoke the Policy Configuration dialog box. For more information, see [Policy Configuration Dialog Box](../core/policy-configuration-dialog-box.md).|  
|**Report to Analyst** property|Select **True** if you want to make this shape viewable in the Visual Business Analyst Tool.|
