# Tracking Profile Editor UI Help
This section provides instructions for the Tracking Profile Editor user interface (UI). You access this information by pressing the F1 key or clicking **Help** in the UI. Use the Tracking Profile Editor to extract the data the business analyst requires and to perform the mapping between the specific business event data and the actual orchestration. For general information about the Tracking Profile Editor, see [Using the TPE](../core/using-the-tpe.md).
