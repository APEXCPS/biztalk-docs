# Configuration Wizard, Unconfigure Features Page
Use the **Unconfigure Features** page to remove features from the BizTalk Server configuration.  
  
> [!NOTE]
>  The list only contains features that are already configured.  
  
|Use this|To do this|  
|--------------|----------------|  
|**Features to be unconfigured**|Select the features you want to unconfigure on this computer, and then click **OK**.|
