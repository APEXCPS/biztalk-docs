# Configuration Wizard, Start Page
Use the **Start** page to configure your server when using a default or custom configuration.  
  
|Use this|To do this|  
|--------------|----------------|  
|**Basic configuration**|This option allows you to configure the server with default settings. **Caution:**  If databases exist on this server, you might need to switch to manual configuration.|  
|**Custom configuration**|This option allows you to configure the server using advanced configuration options. Use this option to enable and disable features and customize the values for each database and service account.|  
|**Database server name**|Type the name of the computer where SQL Server is installed. If you select the default configuration option, all databases will be created on this server. If you select the custom configuration option, this value will be used as the default value for all databases which you can later customize.|  
|**User name**|Type the user name for the account that the BizTalk Server services will run under.|  
|**Password**|Type the password for the account that the BizTalk Server services will run under.|
