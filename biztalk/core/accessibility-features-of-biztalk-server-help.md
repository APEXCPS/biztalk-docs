# Accessibility Features of BizTalk Server Help
BizTalk Server Help can be made more accessible for people with disabilities. For example, you can customize the font size and color of dialog boxes and browser settings. The following procedure explains how to find this accessibility information in Microsoft Document Explorer, the user interface in which BizTalk Server Help is displayed.  
  
## Finding information about accessibility options for Help  
  
#### To find information about accessibility options for BizTalk Server Help  
  
1.  On the **Help** menu, click **Help on Help**. The **Help on Help** page for Microsoft Document Explorer is displayed.  
  
2.  On the **Help on Help** page, click **Customizing Microsoft Document Explorer**.  
  
3.  On the **Customizing Microsoft Document Explorer** page, click **Accessibility and Microsoft Document Explorer**.
