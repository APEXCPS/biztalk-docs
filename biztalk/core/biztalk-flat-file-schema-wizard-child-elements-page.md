# BizTalk Flat File Schema Wizard, Child Elements Page
Use the **Child Elements** page to specify the properties of the child elements in the record.  
  
|Use this|To do this|  
|--------------|----------------|  
|**Element Name**|Enter the name for the element.|  
|**Element Type**|Select the element type from the drop-down selection box. To change the element type for multiple child elements, you can select all and then right-click.|  
|**Data Type**|Select the data type from the drop-down selection box.|  
|**Contents**|This column displays the content for the element. You cannot modify this content.|
