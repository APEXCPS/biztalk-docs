# Business Rule Composer UI Help
This section provides instructions for the **Business Rule Composer** user interface (UI). You access this information by pressing the F1 key or clicking **Help** in the UI. Use the **Business Rule Composer** to create and modify policies and vocabularies. For more information about the Business Rule Composer, see [Creating and Using Business Rules](../core/creating-and-using-business-rules.md).
