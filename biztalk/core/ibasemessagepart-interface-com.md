# IBaseMessagePart Interface (COM)
Describes a part belonging to one or more messages.  
  
 For a list of all members of this type, see [IBaseMessagePart Members](../core/ibasemessagepart-members-com.md).  
  
> [!NOTE]
>  For information on using this interface within managed code, see the `IBaseMessagePart Interface`.  
  
## Remarks  
 A message is composed of one or more parts, and a part can be used in more than one message.
