# Configuration Wizard, Consolidated Databases View Page
Use the **Consolidated Databases View** page to view an aggregated list of features, servers, databases, and data stores used in your configuration.  
  
|Use this|To do this|  
|--------------|----------------|  
|**Edit**|Edit the database server name and database name of the selected features. **Note:**  You can edit multiple features by pressing CTRL and clicking the mouse.|
