# DeleteSNAService
The **DeleteSNAService** function deletes a particular service using the Service Control Manager. All the keys for the service are deleted as well.  
  
## Parameters  
 *Argument 0*  
 Name of the service to be deleted.  
  
## Return Values  
 *Return 0*  
 Status of the operation.
